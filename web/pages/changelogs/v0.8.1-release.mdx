Welcome to **HyperNote v0.8.1**! This release focuses on improving configuration management and user experience.

## ⚙️ Configuration Management

- **Refactored AppConfig**: Improved `AppConfig` and `AIModel` structures for better maintainability and flexibility.
- **Streamlined Settings**: Enhanced settings command access for easier user configuration.

## 🔒 Subscription System

- **Device-Based Verification**: Implemented device-based subscription verification using Hardware UUID for better security.

## 🎨 UI Improvements

- **Settings Overhaul**: Completely refactored `SettingsView` and related components for improved UI consistency and functionality.
- **Recent Notes**: Streamlined summarization logic in `RecentNotesListView` for better UI responsiveness.
- **Simplified Authentication**: Removed legacy user login and pro status check functionality for a cleaner experience.
