**HyperNote v0.8.6** introduces intelligent clipboard monitoring and AI-powered title generation.

## 📋 Clipboard Monitoring

- **MaybeLike Service**: Added new service for clipboard monitoring and relevance checking - intelligently detects when clipboard content might be worth saving.
- **User Settings**: Enhanced clipboard monitoring feature with customizable user settings.

## 🤖 AI Title Generation

- **Auto Titles**: Added automatic title generation feature to `DoubaoAPI` and updated note saving logic to support it.

## 🛠️ Technical Improvements

- **Code Cleanup**: Removed unused Feishu API files for a cleaner codebase.
