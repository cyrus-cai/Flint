**HyperNote v0.9.0** is a major release bringing powerful AI features and important workflow enhancements.

## 🤖 AI Enhancements

- **AI-Generated Summaries**: Enhanced notification system now includes AI-generated summaries, giving you quick insights into your notes at a glance.
- **Refactored AI Input**: Improved AI input preparation for better, more accurate results.

## 🗑️ Note Management

- **Note Deletion**: Implemented full note deletion functionality - you can now delete notes directly from the interface.
- **Updated Notifications**: Updated notification text to better reflect note actions.

## ⚡ System Improvements

- **Enhanced Clipboard Handling**: Improved clipboard handling and event monitoring in `GlobalKeyMonitor` for more reliable operation.
- **macOS Compatibility**: Refactored `MainWindowController` for better compatibility with upcoming macOS releases.
