# Contributing to Flint

Thanks for your interest in contributing to Flint! Here's how to get started.

## Quick Start

1. Fork the repo and clone it
2. Open `Flint.xcodeproj` in Xcode 16+
3. Set your development team under **Signing & Capabilities**
4. Build and run (`Cmd+R`)

**Requirements:** macOS 15+ · Apple Silicon · Xcode 16+

## Ways to Contribute

- **Bug reports** — Open an issue with steps to reproduce
- **Feature requests** — Open an issue describing the use case
- **Code** — Fork, branch, PR. Keep changes focused and small.
- **Documentation** — Improvements to README, docs, or code comments
- **MCP tools** — Ideas or implementations for new MCP server capabilities

## Pull Requests

1. Create a feature branch from `main`
2. Make your changes
3. Test on a clean macOS 15 install if possible
4. Open a PR with a clear description of what and why

## Code Style

- Follow existing Swift/SwiftUI conventions in the codebase
- Keep commits atomic and messages descriptive

## Questions?

Open a [Discussion](https://github.com/cyrus-cai/Flint/discussions) — we're happy to help.
